# Email Marketing Templates - RestOrder

This directory contains email templates for the 14-day trial nurture sequence.

## Email Sequence Overview

- **Day 0**: Welcome email + quick start guide
- **Day 1**: Customer success story
- **Day 3**: Engagement (celebrate quick win)
- **Day 5**: Social proof compilation
- **Day 7**: Value reinforcement + ROI calculator
- **Day 10**: Urgency (limited-time discount)
- **Day 14**: Last chance (trial expiring)

## Usage

These templates can be imported into your email marketing platform:
- ConvertKit
- Mailchimp
- ActiveCampaign
- Drip
- SendGrid

## Personalization Variables

Use these merge tags in your email platform:
- `{{first_name}}` - Customer's first name
- `{{business_name}}` - Restaurant name
- `{{email}}` - Customer's email
- `{{trial_end_date}}` - When the trial expires
- `{{discount_code}}` - Promo code for offers

## A/B Testing Recommendations

Test these elements:
1. Subject lines (curiosity vs. value-based)
2. CTA button text
3. Sending time (9am vs. 2pm vs. 6pm)
4. Email length (short vs. detailed)
5. Personalization level

## Metrics to Track

- Open rate (target: 25%+)
- Click-through rate (target: 10%+)
- Trial-to-paid conversion (target: 60%+)
- Unsubscribe rate (keep below 0.5%)
